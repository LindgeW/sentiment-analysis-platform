系统简介

情感分析(Sentiment analysis)是自然语言处理(NLP)领域的一个任务， 又称倾向性分析，意见抽取(Opinion extraction)，意见挖掘(Opinion mining)， 情感挖掘(Sentiment mining)，主观分析(Subjectivity analysis)等， 它是对带有情感色彩的主观性文本进行分析、处理、归纳和推理的过程， 如从商品评论文本中分析用户对商品的的评价(positive、negative、neutral). 该平台由Bootstrap4 + jQuery3 + Flask搭建，仅限于电商文本评论的情感分析!

开源

方便

用户只需输入评论文本或上传文本文件，本平台将会该你反馈给您想要的答案. 为了方便使用，平台提供参考文档和源码供你学习和使用！

高效
