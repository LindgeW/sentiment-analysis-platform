# sentiment-analysis-platform
基于LSTM的电商评论情感分析平台

## 技术要点：
 + Java前端：Bootstrap4、jQuery
 + Java后台：SpringBoot
 + Python服务: Python3、Flask
 + 模型：LSTM+Keras+TensorFlow
 + 爬虫：selenium
